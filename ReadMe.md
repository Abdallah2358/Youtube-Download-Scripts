# Prerequisites 
## Interpreter
- [python 3](https://www.python.org/downloads/)

## Pip Libraries
- pytube
- requests
- youtube-dl
### Installation Commands
```
pip install pytube
pip install requests
pip install youtube-dl
```

# Usage 
## Download Single Video  
- Run this command in console 
```ps
python dwVid.py
```
- Follow instruction

## Download a full PlayList
- Run this command in console 
```ps
python dwPlayList.py
```
- Follow instruction
