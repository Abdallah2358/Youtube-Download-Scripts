# Prerequisites 
## Interpreter
- [python 3](https://www.python.org/downloads/)

## Pip Libraries
- pytube
- requests
- youtube-dl
### Installation Commands
```
pip install pytube
pip install requests
pip install youtube-dl
```

# Usage 
## Single Video 
```ps
python dwVid.py
```

## PlayList
```ps
python dwPlayList.py
```
